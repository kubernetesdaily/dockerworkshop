---
title: " Docker Compose CLI - Create Command "
description: " Creates containers for a service. "
---


### create docker-compose.yml with following content 

```yml
version: "3.9"
services:
  web:
    build: .
    ports:
      - "8000:5000"
  redis:
    image: "redis:alpine"

```
